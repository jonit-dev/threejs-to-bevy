# ThreeNative V1 Starter

This starter is generated from the maintained V1 template.

## Commands

```bash
pnpm run validate
pnpm run build
pnpm run dev:web
pnpm run dev:desktop
pnpm run verify
```

`pnpm run build` emits the bundle to `dist/game.bundle/`, including
`manifest.json` and `world.ir.json`.
