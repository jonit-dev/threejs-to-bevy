# Prompt: Physics Knockdown

Build a small browser game where the player launches or pushes an object into a
set of standing targets and scores by knocking them down or moving them out of
place. Controls must use WASD, arrow keys, or space/enter. Include a score,
retry path, and visible target set.

Visual bar: launcher or player, projectile or push object, targets, floor, and
score UI must be readable without inspecting code.
