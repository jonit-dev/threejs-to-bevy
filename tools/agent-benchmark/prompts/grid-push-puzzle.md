# Prompt: Grid Push Puzzle

Build a small browser grid-pushing puzzle where the player controls a visible
character with WASD or arrow keys and pushes at least two visible crates onto
matching goal tiles. Crates may be pushed but not pulled, and walls must block
movement. Include visible progress, a clear win state when every crate is on a
goal, and a reset or retry path.

Visual bar: the player, crates, walls, floor grid, goal tiles, progress UI, and
win state must be readable without inspecting code.
