# Game Velocity Kits

This maintained proof artifact covers the promoted kit surfaces used by the
game-development velocity workflow:

- top-down collector
- lane runner
- checkpoint race

The artifact under `artifacts/game-production/kit-proof.json` is consumed by
`pnpm verify:generated-games` as release evidence for kit package, recipe,
asset-role, UI-state, playtest, screenshot, scale, and QA coverage.
