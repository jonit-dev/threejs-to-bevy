# V8 Camera Views Example

Demonstrates multi-camera rendering with follow helpers, split-screen viewports,
render layers, render-to-texture monitors, custom projection metadata, and
screenshot export declarations.

## Build

```bash
pnpm tn -- build --project examples/v8-camera-views
pnpm tn -- validate --project examples/v8-camera-views
```

## Verify

```bash
pnpm verify:v8:camera-views
```

Artifacts are written to `artifacts/v8/camera-views/`.
